# Project Setup Guide

## Deployment

The application has been deployed.

> **Note:** The backend is hosted on a free server using Render, so the API response may be slower than usual, especially during the request.

## Setup Instructions

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clonehttps://github.com/abhijitkumarIN/quess-assesment-erp
cd quess-assesment-erp
```

### 2. Install Dependencies

Install the required Node.js packages:

```bash
npm install
```

### 3. Run the Project (Development Mode)

```bash
npm run dev
```

### 4. Run the Project (Production Mode)

First build the project and then start the server:

```bash
npm run build
npm start
```

## UI Information

The UI for this project is based on an open-source template.
